# ヘルプ

事前Q&Aコーナーです。


## 時刻表が正しく表示されません。

キャッシュの削除をお試しください。

アプリを起動し「設定 > キャッシュ削除 > 削除」を選択します。
キャッシュされた時刻表データなどを削除します。
